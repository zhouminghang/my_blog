1.1 react是什么？

我自己的认识：

  • 首先它是一个JS库
    （是Facebook开源的一个用于构建用户界面的JavaScript库，比较具有代表性的是Facebook及其旗下的Instagram项目）
  • 专注于MVC架构中的V,即视图层
    （与AngularJS的MVC特点形成鲜明的对比，这使得React很容易和开发者已有的开发栈进行融合）
  • 以组件化开发为主
    （重复利用性高，可组合使用）
  • 口号：Learn Once, Write Anywhere（学习一次，到处使用）
    （jQuery口号：Write Less, Do More写的更少，做的更多，有点像二八原则，即以20%的代码实现80%的功能）

AngularJs之路由篇

ng-router / ui-router

ng-router 与 ui-router 的异同点：

区别 ng-router ui-router
名称 ng自带路由 第三方路由
功能 能力有限，不能嵌套路由 非常强大，可以循环嵌套路由，实现路由的深层嵌套
下载安装 bower install –save-dev angular-router @x.y.z(版本号，不写默认最新版本) bower install –save-dev angular-ui-route
引入的库 angular-route.min.js angular-ui-router.min.js
占坑标签 <div ng-view></div> <div ui-view></div>
单页面标签 需要加ng-controller=”” 不需要加ng-controller=””
注入依赖模块 [“ngRoute”] [“ui.router”]
相同点 都依赖angularjs库，引到ng后，做单页面应用的时候存在跨域，需要开启服务器
