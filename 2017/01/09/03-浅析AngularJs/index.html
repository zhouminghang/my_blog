浅析AngularJS（第一篇）

最近在学习Angularjs，有兴趣的童鞋可以和我一起学习，今天更新这篇博客，仅供参考，有不懂得小伙伴可以上官网，查看官方文档
MOre info: AngularJs

一.AngularJs是什么？

Angularjs 简称ng，它是一款非常优秀的前端高级框架。

1.1 为什么说它高级呢？

angularjs类似于jquery，它基于原生js进行一系列的封装，jQuery致力于大量操作DOM元素，而Angularjs最大限度的避免了操作DOM元素，开发者只需要关心页面上显示什么样的数据即可，大大的提高了开发效率，AngularJs 使得开发现代的单一页面应用程序（SPAs：Single Page Applications）变得更加容易。

1.2 说说前端最火的三大框架？

  • No1： vue.js 国内阿里公司开发 精而美
  • No2： react.js Facebook公司开发
  • No3： angular.js 谷歌公司 大但比较全面而笨重

1.3 angularjs历史：

  • 09年被谷歌公司收购，解决了开发者大量操作Dom元素的问题，重心放在了数据上，极大地提高了开发效率；
  • angularjs的版本：
    • angularjs 1.x 体积非常庞大，大概几M，但是比较全面，这样的后果是： 非常笨重，影响性能，也由于angularjs 最初的框架设计的问题，导致它的性能确实一般；
    • angularjs 2.x 更新速度非常慢，正式版迟迟不发布，16年9月，才发布正式版本；
    • 而且 2.x 版本 使用 typescript（微软发明）编写，而非javascript，全面拥抱了ES6的语法，遵守ES6de语法规则；

1.4 angularjs的特性

  • MVC/MVVM
    • 它是一种开发思想，而不是一种设计模式；
    • 含义 model（模型），view (视图)， controller （控制器–调度view和model，是两者之间的桥梁）
    • 关于设计模式，请参考 大话十二种设计模式；
  • 模块化
  • 依赖注入
  • 自动化双向数据绑定

2.2 实现最简单的数据双向绑定

  • 先来了解一些基本概念
    • ng-app ：指令，本质是 自定义属性
    • ng-app ：在angularjs中有模块的概念，ng-app就是指明该段html页面受到哪个模块的控制。
    • ng-app在一个页面中尽量只写一个。页面中只有一个ng-app会自动启动，如果有两个必须手工启动（下面会介绍怎么手工启动）。
    • ng-controller ：表明该段代码到底受到这个模块哪个控制器的控制；
  • 整个工程尽量只有一个模块，一个页面只由一个控制器来控制。

三.angularjs总结：

  • AngularJS 把应用程序数据绑定到 HTML 元素。
  • AngularJS 可以克隆和重复 HTML 元素。
  • AngularJS 可以隐藏和显示 HTML 元素。
  • AngularJS 可以在 HTML 元素”背后”添加代码。
  • AngularJS 支持输入验证。
