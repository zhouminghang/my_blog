Linux操作系统下最常用的简单命令：

1.创建文件夹

  • mkdir + 空格 + 文件夹名称
  • mkdir： make-创建、 directory-目录

    举例： 创建一个folderName1的文件夹

  • mkdir folderName1

    2.删除文件夹

  • rmdir + 空格 + 文件夹名称
  • rmdir： remove-移除、 dir-目录缩写

    举例：删除folderName1文件夹

  • rmdir folderName1

    3.创建一个文件

    方法1：使用echo命令

  1. echo '文件里的内容' >> 文件名称
  • 内容必须用’单引’或者”双引”引起来
  • 只写引号，不写内容，内容为空
  • 关于尖括号的区别：

    • 一个尖括号，表示输出到

    • 两个尖括号，表示追加到

      举例： 创建一个test文本文件,内容为空（以.txt格式为例）

      • echo ‘’ >> test.txt

        向test.txt文件追加内容

      • echo ‘哈哈，我是TXT文本1’ >> test.txt

        方法2： 使用touch命令

  1. touch + 空格 + 文件名称

    举例： 创建test2.txt文件

  • touch test2.txt

    4.删除文件

  • rm + 空格 + fileName（文件名称）

    举例：删除上边刚创建的文件

  • rm test2.txt
  • 该命令有提示功能：提示是否删除该文件？ 选择y

    关于rm的参数-rf：

  • rm -rf test2.txt
  • -rf 是参数，不能乱用！！！
  • 功能：强制删除
  • 不会像之前的没加参数的命令，不会提示是否删除，直接强制删除

    不建议，Linux下不建议使用-rf

  • 为什么？
    • 比如rm * -rf
    • 删除所有，而Linux所有的文件都在根目录下，会导致系统崩溃
    • 特别强调！！！不能乱用

5.切换到根目录

  • cd /
  • cd： 进入/切换 到某个目录下
  • /: 左斜杠，在Linux操作系统下表示根目录

    关于cd的使用小技巧：

  • 为了方便返回之前的目录，而不用级级回退或进入
  • 可使用cd -，直接返回上次保留的目录位置

    Linux与windows操作系统下根目录的写法：

  • Linux下不分磁盘，所有的文件都挂载在根目录下，
    • /:左斜杠，代表Linux下的根目录
  • Windows OS 分很详细的磁盘结构，分盘很明确
    • \：使用右斜杠

      6.怎么查看目录

  • ls 命令
  • 显示该目录下的所有文件或文件夹

    7.Linux的工具

  • vi 和 vim 文本编辑器
